# Superstore Sales Analysis

## Project Overview
This project analyzes retail sales data to uncover trends in revenue, profit, and product performance. Using Microsoft Excel, the analysis provides insights into sales patterns and business performance through interactive dashboards and visualizations.

## Objectives
- Analyze sales and profitability trends.
- Identify top-performing products and categories.
- Evaluate regional sales performance.
- Present findings through an interactive dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Data Cleaning

## Key Insights
- Identified high-performing product categories and regions.
- Analyzed sales and profit relationships.
- Highlighted trends and patterns to support business decisions.

## Dashboard Features
- Sales Performance Overview
- Profit Analysis
- Category and Sub-category Breakdown
- Regional Performance Analysis
- Interactive Charts and KPIs
